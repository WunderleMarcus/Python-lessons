## 🛠️ Nutzung von Git

**Dateien hinzufügen**

```bash
git add .
```

**Commit erstellen**

```bash
git commit -m "Beschreibung deiner Änderungen"
```

**Änderungen pushen**

```bash
git push
```

**Beim ersten Push:**

```bash
git push -u origin main
```

**Änderungen aktualisieren**

```bash
git pull
```
